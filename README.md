# Chok Chok Make

Bem-vindo(a) ao repositório do meu **primeiro site**!

Este projeto é uma simulação de uma loja online de maquiagem totalmente inspirada na **estética coreana (K-Beauty)**. O design foi pensado para ser bem clean, delicado e utiliza uma paleta de cores em tons de rosa.

---

## 📌 O que o projeto tem?

O foco principal foi aprender a estruturar e estilizar uma página do zero. O site conta com uma página única contendo:
*   **Header (Cabeçalho):** Com o nome da loja e um menu de navegação.
*   **Vitrine de Produtos:** Uma seção mostrando alguns produtinhos de maquiagem com um visual limpo e organizado.
*   **Footer (Rodapé):** Final da página com os direitos autorais e informações fictícias.

---

## 🖱 Tecnologias Aprendidas

Como este foi o meu ponto de partida no desenvolvimento web, utilizei:
*   **HTML5:** Para criar toda a estrutura do site.
*   **CSS3:** Para dar vida ao projeto, aplicando os tons de rosa, organizando os elementos e criando a estética clean.
